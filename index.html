Guia de Referência de Testes

Quando usar cada teste estatístico

Comparação de grupos
🔵 Teste t independente
2 grupos independentes, dados normais e variâncias iguais. N ≥ 5 por grupo.
🟢 ANOVA (um fator)
3+ grupos independentes, dados normais, variâncias homogêneas. Seguir com Tukey/Bonferroni.
🟠 Mann-Whitney (U)
2 grupos, dados não normais ou ordinais. Alternativa não parametrica ao teste t.
🔴 Kruskal-Wallis
3+ grupos, dados não normais. Alternativa não parametrica a ANOVA. Seguir com Dunn.
Correlação e Regressão
📈 Correlação de Pearson (r)
Relacao linear entre 2 variáveis continuas normais. -1 ≤ r ≤ 1. Interpretação: |r| > 0.7 = forte.
📊 Correlação de Spearman (rho)
Relacao monotônica entre 2 variáveis. Dados ordinais ou não normais. Robusto a outliers.
📉 Regressão linear simples
Prever Y a partir de X linear. Requer normalidade dos resíduos e homocedasticidade.
