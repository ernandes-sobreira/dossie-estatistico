Entrada de Dados

Cole ou importe seus dados no formato Grupo, Valor
A, 23.1 | A, 24.0 | B, 21.4
Para correlação: Grupo, X, Y. Para bolhas: Grupo, X, Y, Tamanho.
Identifique suas variáveis para facilitar a interpretação dos gráficos e análises.
Modelo 1: comparação simples (teste t, Mann–Whitney). Modelo 2: múltiplos grupos (ANOVA, Kruskal–Wallis, post-hoc). Modelo 3: duas variáveis contínuas (correlação, regressões).

Verificação de Premissas

Antes de escolher o teste, verifique se seus dados atendem às premissas estatísticas
📐NormalidadeAguardando

⚖️Homocedasticidade (Levene)Aguardando
Verifica se as variâncias dos grupos são semelhantes (premissa para ANOVA e teste t).
